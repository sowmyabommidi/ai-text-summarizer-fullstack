# AI Text Summarizer

A full-stack AI text summarizer built with React and Node.js.

## Features
- Text summarization
- Key point extraction
- Sentiment analysis
- AI powered (OpenRouter API)

## Tech Stack
- React (Vite)
- Node.js
- Express
- OpenRouter API

## Installation

### Client
cd client
npm install
npm run dev

### Server
cd server
npm install
node src/index.js

## Environment Variables
Create .env file in server folder:

OPENROUTER_API_KEY=your_api_key_here
